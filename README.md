# MyGram
Simple social media that works like instagram

## Usage

### Start Using It
1. Run the query from [mygram.sql](https://github.com/evrintobing17/MyGram/blob/master/mygram.sql) to make database and table

2. To start the app:
```golang
go run main.go
```
